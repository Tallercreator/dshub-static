# DSHub · Глоссарий — карта визуалов

**121 терминов** (понятия + UI-компоненты). `visualType` определяет, что показывать, и ложится в JSON-рендерер.

## Сводка по типам

| Тип | Сколько | Что это |
|---|---|---|
| `component` | 21 | Живой UI-виджет показывается как есть (кнопка, селект, модалка…) |
| `interactive` | 23 | Живое взаимодействие: тоггл, слайдер, табы, Песочница токенов, контраст-чекер |
| `anatomy` | 8 | Модуль Анатомия — части/состояния (Кнопка, Инпут, Карточка, Cell, Компонент, Атом) |
| `diagram` | 26 | SVG-схема (потоки, слои, архитектура) |
| `example` | 23 | Стилизованная иллюстрация + ссылка на кейс |
| `text` | 20 | Только определение + категория + «встречается в» |

**Важно:** `component`-термины — это твои же атомы/молекулы из инвентаря страниц. Собрал компонент → получил и его демо в глоссарии. Модули Анатомия и Песочница обслуживают группы `anatomy`/`interactive`.
